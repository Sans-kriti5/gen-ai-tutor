# 📚 AI Academic Tutor

An AI-powered academic assistant that answers questions from study materials using **Google Gemini API** and **Retrieval-Augmented Generation (RAG)**.

---

## 🚀 Features

- 📄 Upload and query study materials
- 🤖 AI-powered responses using Gemini
- 🔍 Retrieval-Augmented Generation (RAG)
- 💬 Interactive Streamlit interface
- ⚡ Fast semantic search with FAISS

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- LangChain
- FAISS
- PyPDF

## ⚙️ Installation

```bash
git clone https://github.com/Sans-kriti5/gen-ai-tutor.git
cd gen-ai-tutor
pip install -r requirements.txt
```

Create a `.env` file:

```env
GOOGLE_API_KEY=your_api_key_here
```

Run the application:

```bash
streamlit run app.py
```
